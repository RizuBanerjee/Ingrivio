Create a modern, production-ready cross-platform mobile application called Ingrivio (Android + iOS) using Flutter or React Native with a scalable backend architecture.

App Overview

Ingrivio is an AI-powered nutrition, recipe, and meal-planning platform that helps users:

Identify ingredients from photos.
Suggest recipes based on available ingredients.
Generate detailed cooking instructions.
Recommend high-quality recipe videos.
Analyze food nutrition from images.
Track calories and nutrition like SnapCalorie.
Chat with an AI nutrition assistant.
Plan meals through an integrated calendar.
Access premium AI-powered features through subscriptions.

The UI should be modern, premium, responsive, visually appealing, and suitable for App Store and Google Play release.
